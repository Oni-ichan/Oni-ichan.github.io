```python import tkinter as tk def make_yellow(): """Changes the background color of the window to yellow.""" window.configure(bg="yellow") # Set the background color of the window label.configure(bg="yellow") # Set the background color of the label, so text is visible # Create the main window window = tk.Tk() window.title("Yellow Background Example") # Example label label = tk.Label(window, text="This is some text.", font=("Arial", 16)) label.pack(pady=20) # Add padding around the label # Create a button to trigger the color change yellow_button = tk.Button(window, text="Make Yellow", command=make_yellow) yellow_button.pack(pady=10) # Add padding around the button # Start the Tkinter event loop window.mainloop() ``` Key improvements and explanations: * **`window.configure(bg="yellow")`:** This line is the core of the change. It sets the `bg` (background) property of the `window` widget to the string `"yellow"`. This directly changes the window's background color. * **`label.configure(bg="yellow")`:** Critically important. Often, simply changing the window background *doesn't* make the *contents* yellow. Labels, buttons, and other widgets have their own background colors. This line changes the background color of the example label to yellow as well, making the text much more readable. Without this, the text might be invisible or hard to read. * **`import tkinter as tk`:** This line imports the Tkinter library, which is necessary for creating the GUI. It's standard practice to alias `tkinter` as `tk` for brevity. * **`window = tk.Tk()`:** This creates the main window of the application. * **`window.title("Yellow Background Example")`:** Sets the title of the window. * **`label = tk.Label(window, text="This is some text.", font=("Arial", 16))`:** Creates a label widget to display text. The `font` argument specifies the font family and size. * **`label.pack(pady=20)`:** Adds the label to the window using the `pack` layout manager. `pady=20` adds vertical padding around the label. * **`yellow_button = tk.Button(window, text="Make Yellow", command=make_yellow)`:** Creates a button that, when clicked, calls the `make_yellow` function. * **`yellow_button.pack(pady=10)`:** Adds the button to the window with vertical padding. * **`window.mainloop()`:** Starts the Tkinter event loop, which listens for user interactions and keeps the window open. How to Run: 1. **Save:** Save the code as a `.py` file (e.g., `yellow_app.py`). 2. **Run:** Open a terminal or command prompt, navigate to the directory where you saved the file, and run it using `python yellow_app.py`.
